# Annotate Modules

Content modules for [Annotate](https://github.com/al-robertson/learning-shell) — code documentation and guided walkthroughs.

## Structure

```
index.json                          # Module registry
fastapi-reference/
  module.json                       # Lesson definitions + guide content
  files/                            # Bundled source files (point-in-time snapshots)
    database.py
    user_model.py
    user_schema.py
    user_router.py
    server.py
```

## Module Format

Each module is a folder containing:

- `module.json` — metadata, lesson list, and guide markdown for each lesson
- `files/` — source code files referenced by code-view lessons

The top-level `index.json` lists all available modules with their ID, title, subtitle, and description.

## Adding a Module

1. Create a new folder: `my-module/`
2. Add `module.json` with the lesson structure
3. Add any source files to `my-module/files/`
4. Register it in `index.json`

Modules can be created manually, via the Annotate Studio, or generated by AI agents (e.g. as part of a PR workflow).
